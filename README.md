# CP Command Center

Version 7.3.1 — local-first dashboard for commute, weather, news, Reader playback, MPG/fuel tracking, maintenance, health logs, and retirement snapshots.

Live GitHub Pages site: https://pcal3278-blip.github.io/cp-command-center/

## Data

App data is stored in the browser on the current device. Use **Settings → Export JSON** for backups. Existing `cpCommandCenter.v5.2` and compatible fuel history are migrated into the 6.1 data store on first load.
